Qualityprediction-Cremeproduction-
is an application to make it easy to get an overview about the quality of Your cosmetic cream Batch. 
There is a model calculating your parameters (temperature, fat_content, mixing_time, stirring_speed, PH) for a first simple prediction 
on the main page. 
There are 
a link to a heatmap that shows how the parameters determine the quality of a batch, 
a link to a cream expert chatbot that answers further questions and 
a link to a window for a RAG search about information concearning cosmetic cream production. 
How to run it locally: please install flask, pandas, numpy, matplotlib, joblib, json, plotly, openai, dotenv, os to make it run.
